Class ResultsArray

source code

   object --+    
            |    
numpy.ndarray --+
                |
               ResultsArray

A primitive wrapper to allow accessing numpy.ndarrays via named rows and columns. The ResultsArray.row_names and column_names must be assigned after the object is created.

The names will not carry over for any operations.
